Introdução

HTML (HyperText Markup Language) é a linguagem de marcação utilizada para desenvolver páginas web. Ela define a estrutura e o conteúdo de um documento na internet com a utilização de elementos como tags e atributos, permitindo que navegadores interpretem e exibam o conteúdo de forma adequada. HTML é a base de qualquer página web. Sem ele, não seria possível estruturar e apresentar conteúdo na internet. Estando presente em todos os navegadores web. Atualmente se encontra na versão html5 que introduz novos elementos, atributos e comportamentos, além de suporte melhorado para multimídia e gráficos sem a necessidade de plugins adicionais.

Estrutura básica do html

Html

A tag < html > é a primeira a ser utilizada, ela define a estrutura e o conteúdo do documento. Basicamente ela que indica ao navegador que aquilo é um documento em html.

Head

A tag < head > é usada para conter informações meta, links para estilos, scripts e outros elementos que não são exibidos diretamente na página web, mas que são essenciais para o funcionamento e apresentação da página. Para a criação de um site simples podemos utilizar apenas a tag < title > que tem a função de nomear a aba do site.

body

A tag < body > fica responsavel por tudo que é visivel ao usúario dentro do site e onde se localiza a maior parte das tags de um site.

Tipos de tags comuns de serem vistos na web

Tags HTML são comandos usados para criar e estruturar elementos em uma página web. Elas são a base do HTML e definem como o conteúdo deve ser exibido pelos navegadores. Elas são compostas na maioria das vezes por uma tag de abertura <...> e outra de fechamento < /...> com os elementos que deseja aplicar, havendo algumas exceções de tags que não possuem tag de fechamento como a tag < img >.

    Cabeçalho e parágrafo:

    • h1 a h6
      São utilizadas no tamanho da letra principalmente para titulos e subtitulos sendo o h1 o maior e o h6 o menor.
    • p
      É utilizado para iniciar parágrafos.

    formatação de texto:

    • i
      Utilizado para deixar o texto em itálico exemplo.
    • u
      Utilzado para sublinhar o texto exemplo.
    • strong e b
      Utilizado para deixar o texto em negrito exemplo
    • sub
      Utilizado para texto subescrito exemplo.
    • sup
      Utilizado para texto sobescrito exemplo.
    • mark
      • Utilizado como marcação de texto exemplo
    • small
      • Utilizado para diminuir o tamanho da letra , bastante utilizado em rodapé exemplo

    Listas:

    • ul
      Gera uma lista não ordenada.
    • ol
      Gera uma lista ordenada.
    • li
      Gera um item dentro de uma lista sendo ela ordenada ou não ordenada.

    Link e imagens:

    • a
      É uma tag que não possue fechamento e é responsavel por criar links para navegação entre paginas web, arquivos e dentro da própria pagina
    • img
      Utilizada para inserir imagens
